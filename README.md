Getting started

Requirements:

- Node.js (16+ recommended)

Install dependencies:

```powershell
npm install
```

Install Playwright browsers:

```powershell
npx playwright install
```

Run the sample script:

```powershell
node MyfirstScript.js
```

Run Playwright tests (if you add them):

```powershell
npx playwright test
```
